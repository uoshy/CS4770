User Stories

    Roles
      Admin
      Registered User
      • ↳ Student
      • ↳ Marker
      • ↳ Instructor
01: As an admin I want to create accounts so users can access the system.
  • External user authentication?
02: As an admin I want to create courses to organize users into classes.
  • Semester-specific courses
03: As an admin I want to assign registered users to courses.
04: As an admin I want to adminstrate accounts.
  • Remove registered users from courses
  • Reset passwords, etc.
05: As an admin I want to assign markers to a course.
06: As an instructor I want to create new assignments to they can be distrubited to students.
  • Distribute to students. Notifications?
  • Include code skeletons if necessary
  • Modify assignments
07: As an instructor I want to create test suites so that code can automatically be tested.
08: As an instructor I want test suites associated with an assignment to assist in marking assignments.
  • Automatically run tests on submitted assignments
  • Possibility for returning test report to student
09: As an instructor I want to retrieve students' marks.
  • XML, CSV, Excel
  • Should be automatic and easy to obtain classlist or individuals
10: As an instructor I want to upload assignment solutions so users can access the solutions.
  • Should be associated with a particular assignment
11: As a marker I want to mark student assignments.
  • Download source code
  • Commenting source code, tagging lines of source code
  • Compiling/Running student code
  • Upload separate files for comments
12: As a student I want a text editor so that I can write code.
  • Find/Replace
  • Syntax highlighting/Bracket matching
  • Indentation
13: As a registered user I want to compile code through the cloud.
  • Supports Java, C, C++ (maybe others)
  • Return compiler output
  • Download compiled class/executable files
14: As a registered user I want to download compiled object files so I can execute written code.
  • Java, C, C++, etc.
15: As a registered user I want to upload/download files to share files with other users.
  • Support multiple files at once/archives
16: As a registered user I want to execute command line applications to test written code.
  • Code execution on cloud
  • Terminal-type window
  • Input/Output supported
17: As a student I want to submit assignments so that they can be marked.
18: As a registered user I want to run GUI applications
  • Applets in browser?
  • Return an executable jar/exe for running on client
19: As a student I want to run tests on my code
  • Test suites created by instructor
  • Don't have to submit code to run test suites on it
20: As a registered user I want to create files
  • Associated or unassociated with an assignment
  • Text editor is hub for creation
21: As a student I want to view feedback on my assignment submissions
  • Feedback paired to assignment submissions
  • Can be in-line in assignment code or in separate file
