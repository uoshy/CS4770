User Stories

    Roles
      Admin
      Registered User
      • ↳ Student
      • ↳ Marker
      • ↳ Instructor
  1. As an admin I want to create accounts.
  2. As an admin I want to create courses.
    • Assign instructors to courses
    • Semester-specific courses
    • Assign students/accounts to courses
  3. As an admin I want to adminstrate accounts.
    • Remove accounts from courses
    • Reset passwords, etc.
  4. As an instructor I want to assign markers to a course.
  5. As an instructor I want to create new assignments.
    • Distribute to students. Notifications?
    • Include code skeletons if necessary
  6. As an instructor I want test suites associated with an assignment.
    • Automatically run tests on submitted assignments
    • Possibility for returning test report to student
  7. As an instructor I want to retrieve students' marks.
    • XML, CSV, Excel
    • Should be automatic and easy to obtain classlist or individuals
  8. As an instructor I want test suites associated with an assignment.
    • Automatically run tests on submitted assignments
    • Possibility for returning test report to student
  9. As an instructor I want to upload assignment solutions
    • Should be associated with a particular assignment
  10. As a marker I want to mark student assignments
    • Download source code
    • Commenting source code, tagging lines of source code
    • Compiling/Running student code
    • Upload separate files for comments
  11. As a student I want to use a text editor to write code.
    • Find/Replace
    • Syntax highlighting/Bracket matching
    • Indentation
  12. As a registered user I want to compile code through the cloud.
    • Supports Java, C, C++ (maybe others)
    • Return compiler output
    • Download compiled class/executable files
  13. As a registered user I want to upload/download files.
    • Support multiple files at once/archives
  14. As a registered user I want to execute compiled code (command line)
    • Code execution on cloud
    • Terminal-type window
    • Input/Output supported
  15. As a registered user I want to run GUI applications
    • Applets in browser?
    • Return an executable jar/exe for running on client
