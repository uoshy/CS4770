Relational Database

A relational database is organized as a collection of tables. A table is a set of relations. A relation is a n-tuple of values. Normally each value is restricted to being an element from a standard set (i.e., integers, reals, text strings).

More formally, a relation is a subset of the cartesian product of sets that form the ordered list.

The main motivation for using a database in 3715 is the database's ability to manage concurrent access without failures.

The sqlite3 command provides access to the Sqlite3 database.
