Install Oracle Java-8

Some linux distributions already contain java 8.

  1. Google java 8 download
  2. Select the Java SE Development Kit 8 - Downloads - Oracle
  3. Click accept license aggreement and choose your platform (windows, linux, mac).
  4. For 64-bit linux, choose jdk-8u31-linux-x64.tar.gz, or the latest version.
  5. Assuming you are in your home directory. Make an install directory: mkdir local
  6. Extract with the command: tar xvf ~/Downloads/jdk-8u31-linux-x64.tar.gz -C local
  7. Optionally fetch the API documents by googling jdk8 api download. The install is similar to the JDK, except unzip is used to extract the download.
  8. Modify your .bashrc by adding the following to the end of the file: 
    export PATH=$HOME/local/jdk1.8.0_31/bin:$PATH
  9. Logout and login again to update your environment.
  10. Check that you are accessing the correct java, with the command: 
    % java  -version
java version "1.8.0_31"
Java(TM) SE Runtime Environment (build 1.8.0_31-b13)
Java HotSpot(TM) 64-Bit Server VM (build 25.31-b07, mixed mode)

Install Apache Ant

  1. Google install apache ant, click on the link http://ant.apache.org/manual/install.html. Following the instructions. The details are given below.
  2. Follow the link: http://ant.apache.org/bindownload.cgi, and download the zip archive: apache-ant-1.9.4-bin.zip
  3. Change into local directory, and use the command: 
    unzip ~/Downloads/apache-ant-1.9.4-bin.zip
  4. Modify your .bashrc by adding the following to the end of the file: 
    export PATH=$HOME/local/apache-ant-1.9.4/bin:$PATH
export ANT_HOME=$HOME/local/apache-ant-1.9.4

An earlier version of ant can be installed on ubuntu 14.04 with: 

sudo apt-get install ant

Install Apache Ivy

  1. Go to http://ant.apache.org/ivy/, and select the download icon.
  2. Download apache-ivy-2.4.0-bin.zip, or latest version.
  3. For installation instructions go to http://ant.apache.org/ivy/history/latest-milestone/install.html.
  4. The most relavant part is:
    Download the version you want here, unpack the downloaded zip file wherever you want, and copy the ivy jar file into your ant lib directory (ANT_HOME/lib).
    Since we do not care about most of the contents, we can use /tmp directory. The commands are: 
    cd /tmp
unzip ~/Downloads/apache-ivy-2.4.0-bin.zip
cd apache-ivy-2.4.0
cp ivy-2.4.0.jar ~/local/apache-ant-1.9.4/lib

An early version of ivy can be installed on ubuntu 14.04 with: 

sudo apt-get install ivy
